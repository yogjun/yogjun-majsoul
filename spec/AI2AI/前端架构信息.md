# 前端架构信息

## 目录结构

```
src/
├── components/           # 可复用组件
├── views/               # 页面组件
├── router/              # 路由配置
├── store/               # 状态管理
├── assets/              # 静态资源
├── utils/               # 工具函数
└── main.ts              # 应用入口
```

## 核心模块

### Dashboard模块
- **位置**: src/views/Dashboard.vue
- **功能**: 麻将游戏数据统计展示工作台
- **特性**:
  - 展示总对局数、胜率、平均分数、总玩家数、活跃玩家、峰值在线等统计数据
  - 展示总营收、平均游戏时长、峰值同时在线等扩展统计数据
  - 提供数据变化趋势指示
  - 提供数据摘要卡片展示关键指标
  - 提供图表展示（玩家在线趋势、游戏对局趋势）
  - 支持数据筛选功能（时间范围、数据类型）
  - 支持数据刷新功能
  - 使用ECharts实现交互式数据可视化
  - 响应式布局设计

### 状态管理
- **位置**: src/store/dashboard.ts
- **功能**: 管理dashboard模块的状态和数据
- **特性**:
  - 提供统计数据的状态管理
  - 实现模拟数据生成逻辑，生成更真实的麻将游戏数据
  - 支持数据初始化和更新
  - 管理筛选状态

### 数据生成
- **位置**: src/utils/dataGenerator.ts
- **功能**: 生成模拟统计数据
- **特性**:
  - 生成符合麻将游戏特点的统计数据（胜率20-60%，游戏时长10-40分钟等）
  - 生成时间序列数据，模拟真实日波动和周末高峰
  - 支持不同时间范围的数据生成
  - 提供灵活的数据筛选功能

## 技术栈
- Vue 3 + TypeScript
- Pinia (状态管理)
- Vue Router (路由管理)
- ECharts (数据可视化)
- Vite (构建工具)
- SCSS (样式处理)
